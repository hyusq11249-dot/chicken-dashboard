## 섹션 3 — 채널별 성과 분석

### 키 메시지

| 채널 | 매출액 | 비중 |
|------|--------|------|
| 배달앱 | 81,404,656원 | **55.2%** |

**핵심 인사이트:**
- 배달앱 1개 채널에 55% 집중 → 플랫폼 수수료·정책 변화에 취약
- 자사앱 비중 11%: 자사앱 리텐션·프로모션으로 수수료 절감 여지
- 포장 20%: 매장(홀) 13%보다 높음 → 포장 고객 자사앱 전환 기회

## 섹션 4 — 지역별 매출 분포

### 키 메시지
> **수도권(서울+경기)이 전체 53.7% 집중. 건당 매출은 충청권 1위 — 전라권은 건당 최저, 매출 효율 개선 필요.**

### 근거 데이터

| 지역 | 매출액 | 비중 | 건당 매출 | 비고 |
|------|--------|------|-----------|------|
| 서울 | 41,116,001원 | 27.9% | 16,546원 | |
| 경기/인천 | 38,059,044원 | 25.8% | 16,462원 | |
| 경상권 | 30,715,642원 | 20.8% | 16,923원 | |
| 충청권 | 18,501,501원 | 12.6% | **17,243원** | 건당 최고 |
| 전라권 | 12,264,181원 | 8.3% | **14,993원** | 건당 최저 |
| 강원/제주 | 6,715,187원 | 4.6% | 16,260원 | |
| **합계** | **147,371,556원** | 100% | 16,529원 | |

**핵심 인사이트:**
- 수도권 집중도 53.7%: 지방 채널 확장 여지
- 충청권 건당 17,243원 vs 전라권 14,993원: **객단가 격차 2,250원** → 전라권 업셀링 전략 필요
- 강원/제주 4.6%: 소규모이나 건당 매출은 평균 수준 → 인지도 확장 검토

## 섹션 7 — 종합 인사이트 & 액션 아이템

### 키 메시지
> **1억 4,737만 매출의 구조적 강점과 약점이 동시에 보인다. 3가지 집중 과제로 다음 분기 목표 1억 6,000만 도달 가능.**

### 핵심 발견 요약

| 구분 | 내용 |
|------|------|
| **강점** | 크리스피·레드양념 등 메인 제품 고른 수요, 11월 반등 성공 |
| **약점** | 배달앱 55% 채널 집중, 순살치킨 3개월 연속 하락 |
| **기회** | 충청권 객단가 최고, 자사앱 전환 여지, 사이드 세트화 |
| **위협** | 배달 플랫폼 수수료 인상, 일요일/월요일 수요 침체 |

### 권장 액션 아이템 3가지

1. **순살치킨 부활 프로모션** — 텐더세트·팝콘치킨 1+1 이벤트로 하락세 반전 시도 (예상 효과: 순살 매출 15% 회복)
2. **자사앱 전환 캠페인** — 포장 고객 대상 자사앱 첫 주문 할인으로 플랫폼 수수료 절감 (배달앱 의존도 55% → 50% 목표)
3. **전라권 객단가 개선** — 최저 건당 14,993원 → 충청권 수준(17,000원) 달성 위한 세트메뉴·업셀링 스크립트 도입
